# Orion Host Discretion

Host discretion is managed in Orion Board. <RoleBadge role="Event Committee" badgeIcon="server_committee_role_icon.webp" color="#db1cb8" /> create and manage entries from the dashboard.

**How to access:** Mod Logs → switch to **Event** → **Host Discretion** tab. Use the Create button to add new entries.

:::info Attachments: Discord vs dashboard
The <code>/host-discretion</code> slash command allows **only 1 attachment at a time**. Use the dashboard to add up to 50 attachments if you need more evidence.
:::

<TextWithButton
  text="Open Host Discretion in Orion Board:"
  buttonLabel="Open Host Discretion"
  buttonHref="https://orion.theedenapis.com/dashboard/event/host-discretion"

For the full host discretion policy, requirements, and how to request additions, see the [Host Discretion List](/docs/event-staff-handbook/Hosts/host-discretion-list) page in the Event Staff Handbook.

---

## Rules

- Same substantial-reason requirement as before
- One entry per host–user pair; updating an existing pair updates the entry

---

## Access

- **Event team and higher** (committee) can see names
- **Only event committee and admins** can see reasons and details
